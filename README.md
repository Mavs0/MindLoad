# MindLoad

MindLoad é uma aplicação web desenvolvida para **estimar padrões de sobrecarga cognitiva** por meio de micro-testes comportamentais digitais.

O sistema coleta métricas de **tempo de reação**, **memória de trabalho** e **atenção sustentada** para gerar um índice proprietário chamado **Cognitive Load Score**.

O projeto foi inspirado em estudos sobre memória de trabalho de **Alan Baddeley**.

⚠️ **O sistema não possui finalidade diagnóstica médica.**

---

## 🚀 Tecnologias Utilizadas

**Frontend:**

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Recharts

**Backend:**

- Firebase
- Firebase Authentication
- Firestore

---

## 📦 Instalação

1. Clone o repositório.

2. Instale as dependências:

   ```bash
   npm install
   ```

3. Rode o projeto:
   ```bash
   npm run dev
   ```

Acesse [http://localhost:3000](http://localhost:3000).
